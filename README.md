# 🚪 Door of Luck Game

A simple Python mini game where the player chooses one of three doors.

## 🎮 How It Works

- The computer randomly selects one correct door.
- The player chooses Door 1, 2, or 3.
- If the selected door is correct, the player escapes.
- Otherwise, the player gets trapped.

## 🛠 Technologies Used

- Python
- random module

## ▶️ How to Run

```bash
python main.py
```
